# Open Source Builder Kit Examples

These examples are demo materials for future CLI and documentation work. They use realistic community-maintenance scenarios, but they do not describe actual usage of this repository or any real project.

## Contents

- `project-profiles/`: example project manifests that describe community goals, maintainers, governance, contribution paths, and health-signal inputs.
- `sample-outputs/`: representative reports the toolkit could generate from those manifests and demo inputs.
- `sample-roadmap.md`: an example community roadmap that shows how maintainers could turn health findings into practical work.
- `../data/demo-inputs/`: structured demo inputs intended for future CLI parsing tests.

## Demo Scenario

The examples model two fictional open-source communities:

- **Civic Data Commons**: a municipal open-data tooling project with volunteer maintainers and public-sector users.
- **Learning Lab Toolkit**: an education-focused documentation and curriculum project supported by teachers and part-time maintainers.

All names, metrics, and project activity are illustrative.
